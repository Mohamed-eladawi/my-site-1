About This Project

Project Overview

This project is a Tech Jobs Market Dashboard built for a Data Acquisition assignment. It collects job data using web scraping, processes the dataset, and visualizes it in an interactive dashboard.

Technologies Used

  • Python Web Scraping (Requests + BeautifulSoup)
  • JSON Dataset Processing
  • HTML + CSS Dashboard UI
  • JavaScript Data Visualization
  • Chart.js for charts
  • GitHub Pages for deployment

Features

  • Interactive Dashboard
  • Analytics Charts
  • Dataset Explorer
  • Search & Filtering
  • Pagination
  • Export to Excel & PDF
